Welcome to Virtual Robotics

If you’re here, you’re probably looking for some documentation for the Virtual Robotics competition, specifically, the CTF (Capture the Flag) edition!

1 Getting Started

  • Download DrRacket from here and install it.

  • You can get the code from github. The best way to do that is to use git. If you don’t know how, ask for help!

  • Open DrRacket

  • Open the files. In DrRacket, click file -> open (on the top) and navigate to VirtualRobotMania/manias/. Then open up the example in "manias/ctf/together.rkt" from inside DrRacket.

  • Install packages. There are two racket packages you need: csexp and rfc6455. If you click the run button on the top right, you’ll get an error, with an embedded link offering to update the racket package directory. Click on it! Then try again, and it will offer you the chance to install a missing package. Keep on doing this until they’re all installed.

  • Start the game. Press run button in the top right. That should launch the game in a web-browser tab. The robots won’t be doing anything too interesting yet, but now you have everything you need to work on your robots. Go here to learn more about the game.

4 Geometry

For the more advanced players, it will likely be helpful to create a more precise description of the physical geometry of the game.

In order to make that slightly more possible, we have provided some code that you can trust works. That way, it will be easier to make progress and you won’t feel like you are simply walking in circles.

Documentation for using this code can be found here.
