Welcome to Virtual Robotics

If you’re here, you’re probably looking for some documentation for the Virtual Robotics competition, specifically, the CTF (Capture the Flag) edition!

1 Getting Started

  • Download DrRacket from here and install it.

  • You can get the code from github. The best way to do that is to use git. If you don’t know how, ask for help!

  • Open DrRacket

  • In DrRacket, click file -> open (on the top) and navigate to VirtualRobotMania/manias/.

  • Open up the example in "manias/ctf/together.rkt" from inside your copy of the code.

  • Press run button in the top right. That should launch the game in a web-browser tab. Watch the robots battle it out!
